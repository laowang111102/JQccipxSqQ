# 学生管理系统简介 python1211

## 项目概述

学生管理系统是基于 Python 开发的一个便捷、高效的管理工具，旨在帮助学校管理员和教师进行学生信息与成绩的管理。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

### 用户登录

- 支持验证码功能，增强安全性。

### 信息查询

- 根据不同 `Stu` 用户身份，查询相应用户信息。

### 成绩管理

- `Stu` 用户可以查看自己的基本信息和成绩。
- `Root` 用户可以查看所有学生信息，并添加学生成绩。

## 系统角色

- `Stu` 用户：普通学生用户，可查询个人信息和成绩。
- `Root` 用户：管理员用户，拥有查看所有学生信息和添加成绩的权限。

## 运行环境

- 操作系统：Windows/Linux/OS X
- Python 版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装 Python 环境。
2. 安装 MySQL 数据库，并进行配置。
3. 部署后端代码和前端资源。
4. 运行后端服务。
